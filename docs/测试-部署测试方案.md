# 项目服务器部署与测试方案

## 1. 部署架构
- 后端运行于服务器，用户通过服务器前端网站扫码登录自己的 WeChat Bot。
- 前端提供可视化配置界面，支持环境变量（`.env`）的在线编辑与管理。

## 2. 环境变量（`.env`）可视化配置
- 用户可在前端界面以**选项勾选**或**填空**形式配置自己的模型与 API 密钥。
- 填空项需提供**模板示例**（如 API 地址格式、密钥格式），便于用户参考填写。

## 3. 用户仓库管理
- 用户的 Vault 仓库**保存在本地**（或同步至本地），服务器不存储用户私有数据。

## 4. 报告同步
- 服务器仅同步**测试者的用户反馈**，不涉及普通用户的测试数据。
- 增加工单系统。也可以直接返回给开发者工单

## 5. 前端功能调用示例
- 网站前端需提供**全部功能**的调用示例（如 API 请求演示、参数说明、返回结果展示），方便测试者快速上手。

## 6. API 用量统计
- 前端提供**用量统计**看板，展示各 API 的调用次数、消耗配额等数据。
